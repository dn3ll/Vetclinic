# VetClinic App

Приложение для владельцев домашних животных, позволяющее управлять профилем питомцев, записываться на прием к ветеринару и вызывать экстренную помощь.

## Возможности

-  **Управление питомцами:** добавляйте своих любимцев, загружайте фото, указывайте кличку, возраст и тип.
-  **Запись на прием:** планируйте визиты к врачу, выбирайте процедуры, даты и специалистов.
-  **Экстренная помощь (SOS):** быстрый поиск ближайшего ветеринара в критической ситуации.
